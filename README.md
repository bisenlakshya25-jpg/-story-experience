# ✨ Cinematic Story Experience

A cinematic, interactive storytelling website built using **HTML, CSS, and JavaScript**.

The project focuses on creating an immersive emotional experience through smooth animations, elegant transitions, dynamic backgrounds, and modular screen navigation.

---

## 🎯 Features

- 🌌 Animated Aurora Background
- ✨ Dynamic Starfield & Floating Particles
- 🎬 Multi-screen Storytelling Experience
- 💎 Glassmorphism UI
- 📱 Fully Responsive Design
- 🎵 Audio & Interaction Effects
- ⚡ Smooth Screen Transitions
- 🧩 Modular JavaScript Architecture
- 🚀 Optimized Performance

---

## 🛠 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Canvas API
- Web Audio API

## 🎨 Design Goals

This project was built around the idea of creating a storytelling experience rather than a traditional webpage.

Key design principles include:

- Minimal UI
- Smooth cinematic pacing
- Emotional storytelling
- Consistent visual language
- Modular architecture
- Performance-first animations
